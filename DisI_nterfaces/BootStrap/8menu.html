Bootstrap – Navbar y menús responsivos (Web autoexplicativa)

1. Navbar y menús responsivos

La navbar de Bootstrap es uno de los componentes más útiles: crea menús de navegación responsivos con muy poco código.

Veremos: .navbar, .navbar-expand-*, .navbar-brand, .navbar-toggler, .collapse y .nav-link.

2. Estructura básica de una navbar

La estructura mínima usa .navbar y un .container dentro: 

<nav class="navbar bg-light">
  <div class="container">
    <a class="navbar-brand" href="#">Mi Web</a>
  </div>
</nav>
Mi Web
  • .navbar → activa estilos de barra de navegación.
  • .navbar-brand → logo o nombre de la web.

3. Hacerla responsiva: .navbar-expand-*

Con .navbar-expand-* indicas a partir de qué tamaño de pantalla la navbar se muestra expandida (horizontal). Por debajo de ese tamaño, el menú se colapsa.

  • .navbar-expand-sm → expandida desde ≥ 576px.
  • .navbar-expand-md → desde ≥ 768px.
  • .navbar-expand-lg → desde ≥ 992px (muy común).
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  ...
</nav>

4. Botón hamburguesa y menú colapsable

Para que el menú se oculte en móvil, se usa:

  • .navbar-toggler → botón tipo “hamburguesa”.
  • .navbar-toggler-icon → icono dentro del botón.
  • .collapse .navbar-collapse → contenedor que se abre/cierra.
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container">
    <a class="navbar-brand" href="#">Mi Web</a>

    <button class="navbar-toggler" type="button"
            data-bs-toggle="collapse" data-bs-target="#menuPrincipal">
      <span class="navbar-toggler-icon"></span>
    </button>

    <div class="collapse navbar-collapse" id="menuPrincipal">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link active" href="#">Inicio</a>
        </li>
        ...
      </ul>
    </div>
  </div>
</nav>

Fíjate en que el data-bs-target del botón coincide con el id del bloque .collapse.

5. Enlaces de navegación: .nav-link

Dentro de .navbar-nav, cada elemento suele tener:

  • .nav-item en el <li>
  • .nav-link en el <a>
<ul class="navbar-nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">Inicio</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Cursos</a>
  </li>
</ul>
Ejemplo
  • Inicio
  • Cursos
  • Contacto

6. Ejercicio – Navbar colapsable en móvil

Enunciado:

Crea una navbar que:

  • Tenga una marca (.navbar-brand) con el texto “Academia Web”.
  • Tenga enlaces: Inicio, Cursos, Blog, Contacto.
  • Se colapse en móvil (menú hamburguesa).
  • Se vea horizontal en pantallas grandes.

Texto plano (sin formato):

academia web inicio cursos blog contacto en movil menu hamburguesa en escritorio menu horizontal

Solución de ejemplo:

Academia Web
  • Inicio
  • Cursos
  • Blog
  • Contacto

Aprende desarrollo web paso a paso

Cursos prácticos de HTML, CSS, JavaScript y frameworks modernos.

Aquí iría el contenido de la página: listado de cursos, descripción de la academia, etc.

Código usado para la navbar:

<nav class="navbar navbar-expand-lg navbar-dark bg-primary">
  <div class="container">

    <a class="navbar-brand fw-bold" href="#">Academia Web</a>

    <button class="navbar-toggler" type="button"
            data-bs-toggle="collapse" data-bs-target="#navbarAcademia"
            aria-controls="navbarAcademia" aria-expanded="false"
            aria-label="Alternar navegación">
      <span class="navbar-toggler-icon"></span>
    </button>

    <div class="collapse navbar-collapse" id="navbarAcademia">
      <ul class="navbar-nav ms-auto">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Inicio</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Cursos</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Blog</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Contacto</a>
        </li>
      </ul>
    </div>

  </div>
</nav>

El alumnado puede probar a cambiar .navbar-expand-lg por .navbar-expand-md o .navbar-expand-sm para ver cómo cambia el punto en el que se colapsa el menú.
