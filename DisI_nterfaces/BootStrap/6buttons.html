Bootstrap – Botones y Call To Action (Web autoexplicativa)

1. Botones y Call To Action (CTA)

Los botones son esenciales en interfaces reales. Bootstrap ofrece clases para estilos, tamaños y comportamiento responsivo sin necesidad de CSS.

Aquí veremos .btn, variantes como .btn-primary, versiones .btn-outline-*, tamaños (.btn-sm, .btn-lg) y utilidades como .d-block y w-100 para móvil.

2. Estructura básica de un botón: .btn

Para convertir cualquier enlace o botón en un botón con estilo, usa .btn. Después añade una variante de color. 

<a href="#" class="btn btn-primary">Botón principal</a>
<button class="btn btn-secondary">Botón secundario</button>
Botón principal

3. Botones outline (.btn-outline-*)

Estos botones muestran solo el borde. Son muy útiles para acciones secundarias. 

<button class="btn btn-outline-primary">Ver más</button>
<button class="btn btn-outline-danger">Eliminar</button>

4. Tamaños: .btn-sm y .btn-lg

Bootstrap permite variar el tamaño de los botones fácilmente: 

<button class="btn btn-primary btn-lg">Grande</button>
<button class="btn btn-secondary btn-sm">Pequeño</button>

5. Botones de ancho completo para móvil: .d-block + w-100

<button class="btn btn-primary d-block w-100">
  Botón ancho completo
</button>

En móvil los botones horizontales pueden quedar estrechos, por eso usar w-100 crea CTAs más accesibles.

6. Ejercicio: sección Hero con CTA

Enunciado:

Crea una sección tipo "hero" con:

  • Un título destacado (.h1)
  • Un subtítulo (.lead)
  • Un botón grande principal (.btn-lg .btn-primary)
  • Un botón secundario (.btn-outline-light)

Texto plano (sin formato):

Bienvenido a la plataforma definitiva aprende desarrollo web desde cero cursos prácticos y proyectos reales empieza hoy mismo más información

Solución de ejemplo:

Bienvenido a la plataforma definitiva

Aprende desarrollo web desde cero con cursos prácticos y proyectos reales.

Empieza hoy mismo Más información

Código usado:

<div class="hero-demo text-center">

  <p class="h1 fw-bold mb-3">
    Bienvenido a la plataforma definitiva
  </p>

  <p class="lead mb-4">
    Aprende desarrollo web desde cero con cursos prácticos y proyectos reales.
  </p>

  <div class="d-flex flex-column flex-sm-row gap-3 justify-content-center">
    <a href="#" class="btn btn-light btn-lg fw-bold px-4">
      Empieza hoy mismo
    </a>

    <a href="#" class="btn btn-outline-light btn-lg px-4">
      Más información
    </a>
  </div>

</div>
