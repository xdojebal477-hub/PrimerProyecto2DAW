Bootstrap: Guía autoexplicativa del Layout

Esta página explica únicamente lo que tus estudiantes necesitan para entender cómo funciona el layout en Bootstrap: filas, columnas, breakpoints y tamaños.

1. El sistema de rejilla (Grid)

Bootstrap organiza la pantalla dividiéndola en 12 columnas. Toda fila se crea con .row y dentro se colocan elementos con la clase .col o variantes como .col-6. 

<div class="row">
  <div class="col">A</div>
  <div class="col">B</div>
  <div class="col">C</div>
</div>

Las columnas con .col se reparten el espacio a partes iguales:

2. Columnas con tamaño fijo: .col-*

Si quieres que una columna ocupe un número concreto de columnas (de las 12), usas .col-4, .col-6, etc. 

<div class="row">
  <div class="col-4">col-4</div>
  <div class="col-8">col-8</div>
</div>
col-4
col-8

Recuerda: la suma debe dar ≤ 12. Si no llega a 12, el espacio restante se queda vacío.

3. Breakpoints: diseño responsive

Bootstrap adapta el layout según el tamaño de pantalla usando breakpoints. Se combinan con las clases .col-sm-*, .col-md-*, .col-lg-*, etc.

  • sm → ≥ 576px
  • md → ≥ 768px
  • lg → ≥ 992px
  • xl → ≥ 1200px
  • xxl → ≥ 1400px

Si no especificas un tamaño para pantallas pequeñas, las columnas se apilan. 

<!-- En móvil: columnas apiladas
En escritorio (md+): 4 + 8 -->
<div class="row">
  <div class="col-12 col-md-4">4 en md+</div>
  <div class="col-12 col-md-8">8 en md+</div>
</div>

col-12 col-md-8

4. Combinar breakpoints (muy común en DAW/DAM)

Puedes definir diferentes tamaños de columna para cada breakpoint. Ejemplo: móvil → 12, tablet → 6, escritorio → 3. 

<div class="col-12 col-sm-6 col-lg-3">...</div>

En móvil → 4 filas. En tablet (sm) → 2 columnas por fila. En escritorio (lg) → 4 columnas en una sola fila.

5. Col auto y col-auto

.col-auto hace que la columna ocupe solo el espacio necesario según su contenido. El resto del espacio lo ocupan los .col normales. 

<div class="row">
  <div class="col-auto">Auto (según contenido)</div>
  <div class="col">Resto del ancho</div>
</div>
