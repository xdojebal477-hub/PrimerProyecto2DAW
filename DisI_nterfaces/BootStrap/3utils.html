1. Espaciado y utilidades en Bootstrap

En esta página vas a ver cómo controlar el “aire” entre los elementos usando: m-*, p-*, g-*, text-center/text-start/text-end y utilidades flex.

Se asume que ya conoces la rejilla básica (.container, .row, .col-*) y quieres mejorar el aspecto visual de tus layouts.

2. Márgenes y padding: m-* y p-*

Bootstrap usa una escala de espaciado numérico: 0 a 5 (y a veces más). Las clases siguen esta estructura:

  • m = margin (margen exterior)
  • p = padding (margen interior)
  • Direcciones: t (top), b (bottom), s (start), e (end), x (horizontal), y (vertical)

Ejemplos: 

<div class="m-3">Margin en todos los lados</div>
<div class="mt-4">Margin solo arriba (top)</div>
<div class="px-5">Padding en horizontal (izquierda y derecha)</div>
<div class="py-2">Padding en vertical (arriba y abajo)</div>

3. Separación entre columnas: g-* en filas

Dentro de una .row, puedes controlar el espacio entre columnas usando la clase g-* (gap). Esto afecta a la separación horizontal y vertical entre las columnas. 

<div class="row g-0">  <!-- sin espacio entre columnas -->
<div class="row g-3">  <!-- espacio medio -->
<div class="row g-5">  <!-- espacio grande -->

4. Alineación de texto: text-start, text-center, text-end

Estas clases alinean el texto dentro del elemento:

  • text-start → alineado a la izquierda (en lenguas LTR)
  • text-center → centrado
  • text-end → alineado a la derecha
<p class="text-start">Texto alineado a la izquierda</p>
<p class="text-center">Texto centrado</p>
<p class="text-end">Texto alineado a la derecha</p>

5. Flexbox rápido: d-flex, justify-content-*, align-items-*

Bootstrap encapsula las propiedades de Flexbox en clases muy cortas. Primero activas Flexbox con d-flex y luego ajustas alineación:

  • justify-content-start | center | end | between | around → horizontal
  • align-items-start | center | end → vertical
<div class="d-flex justify-content-center align-items-center">
  <div>Elemento 1</div>
  <div>Elemento 2</div>
</div>

6. Mini ejercicio (resuelto)

Enunciado: Maquetar tres cajas con espacios y centrados distintos usando: m-*, p-*, g-*, text-* y utilidades flex.

Objetivo didáctico: que el alumnado pruebe combinaciones de márgenes, padding, alineación de texto y Flexbox para ver cómo cambia el layout.

Solución de ejemplo

En este ejemplo usamos un .row g-4 y cada caja tiene sus propias clases de espaciado/alineación.

Caja 1: texto a la izquierda

Clases usadas:

  • p-3 (padding interno)
  • text-start (texto a la izquierda)
  • mb-2 en el título

Esta caja está pensada para texto tipo párrafo justificado a la izquierda.

Caja 2: contenido centrado

Clases usadas:

  • d-flex
  • justify-content-center
  • align-items-center
  • text-center

Ideal para un mensaje principal o un “call to action” centrado.

Caja 3: elementos separados

Etiqueta Texto en el centro

Aquí combinamos d-flex + justify-content-between para separar los elementos.

Código usado para la fila de las 3 cajas

<div class="row g-4">

  <!-- Caja 1 -->
  <div class="col-12 col-md-4">
    <div class="bg-light border rounded p-3">
      <h4 class="h6 text-start mb-2">Caja 1: texto a la izquierda</h4>
      <div class="demo-box text-start p-3 mb-2">
        <p class="mb-1"><strong>Clases usadas:</strong></p>
        <ul class="mb-0">
          <li><code>p-3</code></li>
          <li><code>text-start</code></li>
        </ul>
      </div>
    </div>
  </div>

  <!-- Caja 2 -->
  <div class="col-12 col-md-4">
    <div class="bg-light border rounded p-3">
      <h4 class="h6 text-center mb-2">Caja 2: contenido centrado</h4>
      <div class="demo-box d-flex flex-column justify-content-center align-items-center p-4 mb-2">
        ...
      </div>
    </div>
  </div>

  <!-- Caja 3 -->
  <div class="col-12 col-md-4">
    <div class="bg-light border rounded p-3">
      <h4 class="h6 text-end mb-2">Caja 3: elementos separados</h4>
      <div class="demo-box d-flex justify-content-between align-items-center px-3 py-2 mb-2">
        <span class="badge bg-primary">Etiqueta</span>
        <span>Texto en el centro</span>
        <button class="btn btn-sm btn-outline-primary">Acción</button>
      </div>
    </div>
  </div>

</div>

Puedes modificar valores de m-*, p-*, cambiar text-center por text-end o jugar con justify-content-* y align-items-* para ver rápidamente el efecto.
