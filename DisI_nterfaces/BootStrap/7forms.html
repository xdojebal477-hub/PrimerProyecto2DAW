Bootstrap – Formularios modernos (Web autoexplicativa)

1. Formularios con Bootstrap

Los formularios son uno de los elementos más usados en desarrollo web. Bootstrap ofrece clases simples para crear entradas limpias, accesibles y responsivas.

Veremos .form-control, .form-label, .input-group, .form-select y .form-check.

2. Campos básicos: .form-control

La clase .form-control convierte inputs y textareas en elementos estilizados automáticamente. 

<label class="form-label">Nombre</label>
<input type="text" class="form-control" />

3. Listas desplegables: .form-select

Para selects más limpios y accesibles: 

<label class="form-label">Curso</label>
<select class="form-select">
  <option>DAW</option>
  <option>DAM</option>
</select>

4. Agrupar elementos: .input-group

Sirve para añadir prefijos/sufijos como iconos, unidades, @usuario, etc. 

<div class="input-group">
  <span class="input-group-text">@</span>
  <input type="text" class="form-control" placeholder="usuario">
</div>
@

5. Checkboxes y radios: .form-check

Usados para términos, preferencias, etc. 

<div class="form-check">
  <input class="form-check-input" type="checkbox">
  <label class="form-check-label">Acepto los términos</label>
</div>

6. Ejercicio — Formulario responsivo con validación visual

Enunciado:

Crea un formulario de registro que incluya:

  • Nombre y apellidos
  • Email
  • Contraseña
  • Curso (select)
  • Aceptación de términos
  • Botón de envío

Debe ser responsivo (2 columnas en pantallas medianas) y tener validación visual básica con .is-valid y .is-invalid.

Texto plano (sin formato):

formulario registro nombre apellidos email contraseña curso daw dam asir aceptar términos enviar

Solución de ejemplo:

Este campo es obligatorio.

Código usado:

<form class="p-4 bg-light rounded">
  <div class="row g-3">

    <div class="col-12 col-md-6">
      <label class="form-label">Nombre</label>
      <input type="text" class="form-control is-valid">
    </div>

    <div class="col-12 col-md-6">
      <label class="form-label">Apellidos</label>
      <input type="text" class="form-control is-invalid">
      <div class="invalid-feedback">Este campo es obligatorio.</div>
    </div>

    <div class="col-12">
      <label class="form-label">Email</label>
      <input type="email" class="form-control">
    </div>

    <div class="col-12">
      <label class="form-label">Contraseña</label>
      <input type="password" class="form-control">
    </div>

    <div class="col-12 col-md-6">
      <label class="form-label">Curso</label>
      <select class="form-select">
        <option>DAW</option>
        <option>DAM</option>
        <option>ASIR</option>
      </select>
    </div>

    <div class="col-12 col-md-6 d-flex align-items-end">
      <div class="form-check">
        <input class="form-check-input" type="checkbox">
        <label class="form-check-label">Acepto los términos</label>
      </div>
    </div>

    <div class="col-12 mt-3">
      <button class="btn btn-primary w-100">Enviar</button>
    </div>

  </div>
</form>
