Bootstrap – Tipografía y Colores (Web autoexplicativa)

1. Tipografía y colores en Bootstrap

Bootstrap incluye clases tipográficas y de color que permiten crear jerarquía visual sin escribir CSS.

Esto es clave ahora que el alumnado empieza a maquetar páginas reales con secciones diferenciadas, títulos claros y texto agradable de leer.

2. Clases tipográficas básicas

2.1. Títulos: .h1….h6

Puedes aplicar niveles de título a cualquier elemento (no solo <h1>) usando clases como .h1 a .h6. 

<p class="h1">Título principal</p>
<p class="h4">Subtítulo</p>
<p class="h6">Texto pequeño</p>

Ejemplo de Subtítulo .h4

2.2. Texto destacado: .lead

Se usa para párrafos introductorios más grandes y legibles.

<p class="lead">Este es un párrafo destacado.</p>

Este es un párrafo destacado usando .lead.

2.3. Grosor del texto: .fw-bold, .fw-light

Controla el grosor del texto sin usar CSS adicional. 

<p class="fw-bold">Texto en negrita</p>
<p class="fw-light">Texto fino</p>

Texto fino (.fw-light)

3. Colores de texto y fondo

3.1. Colores de texto: .text-primary, .text-muted

Los colores ayudan a marcar jerarquía, distinguir categorías o atenuar fragmentos. 

<p class="text-primary">Texto importante</p>
<p class="text-muted">Información secundaria</p>

Texto importante (.text-primary)

Información secundaria (.text-muted)

3.2. Colores de fondo: .bg-light

.bg-light añade un fondo gris muy suave, útil para separar secciones.

<div class="bg-light p-3">
  Contenido con fondo claro
</div>

4. Ejercicio

  • .h1….h6
  • .lead
  • .fw-bold
  • .text-primary, .text-muted
  • .bg-light

guia de productividad 2025 trucos para mejorar tu organizacion diaria estos consejos estan pensados para estudiantes de daw y dam incluye uso de atajos de teclado planificacion semanal y reducir distracciones actualizacion 2025 version revisada
